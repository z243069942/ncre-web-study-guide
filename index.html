📖 核心知识要点
1
DNS与URL解析
▶
  • DNS（域名系统）：完成域名与IP地址之间的映射（注意：不是主机名和域名，也不是域名和HTTP）
  • URL格式：协议://主机名[:端口]/路径[?查询字符串]
  • 示例：http://www.sample.com:8080/login.htm → 协议为 http
⚠️ 样题高频：DNS完成的是域名↔IP映射，答案选C
2
HTTP协议基础
▶
  • HTTP 通常监听 TCP/80端口（HTTPS为443）
  • HTTP是应用层协议，基于TCP/IP
消息类型组成
请求消息开始行（请求行）+ 消息头 + 消息体
响应消息开始行（状态行）+ 消息头 + 消息体
⚠️ 易错点：请求消息的第一行叫"开始行/请求行"，响应消息的第一行叫"开始行/状态行"。两者都叫"开始行"
常用请求方法说明
GET获取资源，参数在URL中
POST提交数据，参数在消息体中
PUT更新资源
DELETE删除资源
状态码含义
200OK，请求成功
301/302重定向
404Not Found，资源未找到
500服务器内部错误
1
HTML文档结构
▶
<!DOCTYPE html> <html> <head> <meta charset="UTF-8"> <title>页面标题</title> <!-- title 显示在浏览器标签栏 --> <link rel="stylesheet" href="style.css"> <!-- 外部CSS放在head --> </head> <body> <!-- 页面内容 --> </body> </html>
⚠️ 外部样式表的引用语句应放在 <head> 标签内，不是 <body>！
  • <title> → 页面标题（不是 <header> 也不是 <caption>）
  • <header> → 页面/区块的页眉，是语义标签
2
超链接与多媒体
▶
<!-- 普通链接 --> <a href="http://example.com">访问</a> <!-- 在新窗口打开 --> <a href="url" target="_blank">新窗口</a> <!-- 电子邮件链接（必须用 mailto:） --> <a href="mailto:abc@example.com">发邮件</a> <!-- 图片 --> <img src="img.jpg" alt="描述" width="100">
target属性值效果
_blank新窗口打开（✅常考）
_self当前窗口（默认）
_top顶层窗口
_parent父框架
✅ 超链接不只有文本，图片、按钮等都可以设为超级链接
3
表单与控件
▶
<form action="login.php" method="POST"> <!-- 文本框 --> <input type="text" name="user"> <!-- 密码框 --> <input type="password" name="pwd"> <!-- 单选框 --> <input type="radio" name="sex" value="m">男 <!-- 复选框 --> <input type="checkbox" name="hobby" value="reading"> <!-- 下拉列表 --> <select name="city"> <option value="bj">北京</option> </select> <!-- 文件上传（限制类型） --> <input type="file" accept=".jpg,.png"> <!-- 提交按钮 --> <input type="submit" value="提交"> </form>
⚠️ 复选框用 type="checkbox"，不是 check；单选框用 type="radio"
4
列表标签
▶
标签说明
<ul>无序列表（Unordered List）
<ol>有序列表（Ordered List）✅建立有序列表选ol
<li>列表项（配合ul/ol使用）
<dl>定义列表
5
块级元素 vs 行内元素
▶
块级元素（占满一行）行内元素（不独占一行）
<div>、<p>、<table>、<h1>~<h6>、<ul>、<ol><a>、<span>、<img>、<input>、<strong>
⚠️ 样题高频：<a> 是行内元素（不属于块元素）
1
CSS基本语法
▶
/* 正确语法：选择器{属性:值} */ body { color: black; } p { font-weight: bold; } /* 错误写法（考试常见陷阱） */ body:color=black /* ❌ */ {body:color=black} /* ❌ */ {body;color:black} /* ❌ */
⚠️ CSS全称：Cascading Style Sheets（层叠样式表），不是 Computer/Creative/Colorful
2
CSS优先级（从高到低）
▶
✅ 正确顺序：行内样式表 > 文档层（内部）样式表 > 外部样式表
优先级类型示例
最高行内样式<p style="color:red">
次高内部样式（文档层）<style> p{color:red} </style>
最低外部样式表<link rel="stylesheet" href="...">
3
常用CSS属性
▶
属性说明示例
font-family字体名称 ✅font-family: Arial
font-size字体大小font-size: 16px
font-weight字体粗细（bold=粗体）font-weight: bold
color文字颜色color: red
text-decoration文字装饰（none=无下划线）a{text-decoration:none} ✅
background-color背景色background-color: #fff
⚠️ 控制字体名称用 font-family，不是 font-type！去掉超链接下划线用 text-decoration:none
4
border-width 边框顺序
▶
✅ 记忆口诀：上右下左（顺时针）
border-width: 上 右 下 左; /* 上10px 右1px 下5px 左20px */ border-width: 10px 1px 5px 20px;
⚠️ 样题第20题：上10 下5 左20 右1 → 正确写法：10px 1px 5px 20px（上右下左）
1
数组创建
▶
// 正确写法 ✅ var aa = new Array("george", "tom", "thomas"); var bb = ["george", "tom", "thomas"]; // 字面量 // 错误写法 ❌ var aa = new Array="george","tom","thomas" var aa = new Array(1:"george",2:"tom") var aa = new Array[3]("george","john","thomas")
2
常用内置对象
▶
对象/方法说明示例
Math.round()四舍五入 ✅Math.round(6.25) → 6
Math.floor()向下取整Math.floor(3.9) → 3
Math.ceil()向上取整Math.ceil(3.1) → 4
Math.max/min最大/最小值Math.max(1,2,3) → 3
new Date()获取当前时间 ✅var d = new Date()
str.sub()下标显示 ✅"2".sub() → H₂O
str.sup()上标显示"2".sup() → H²O
⚠️ 四舍五入必须用 Math.round()，不是 round() 也不是 Math.rnd()

BOM常用对象
▶
属性/方法说明
navigator.appName浏览器名称 ✅
navigator.appVersion浏览器版本
window.location当前URL
window.alert()弹出对话框
window.history浏览历史
⚠️ 获取浏览器名称：navigator.appName（不是 client/browser.navName）

常用事件
▶
事件触发时机
onclick鼠标点击
onmouseover鼠标移入
onmouseout鼠标移出
onfocus获得焦点
onblur失去焦点 ✅（不是 unfocus）
onsubmit表单提交
onchange值改变
⚠️ input控件失去焦点触发 blur 事件（不是 unfocus）
5
控制语句
▶
// break 用法 for(var i=0; i<10; i++) { if(i==5) break; // 退出整个循环 ✅ } switch(x) { case 1: break; // 退出switch ✅ case 2: break; } // break 不能用于 if 语句！❌
⚠️ break 只能用于循环体和switch，不能用于if语句（样题第27题）
6
DOM操作
▶
// 获取元素 var el = document.getElementById("id"); var els = document.getElementsByClassName("cls"); var els2 = document.getElementsByTagName("p"); // 修改内容 el.innerHTML = "<b>加粗文字</b>"; el.innerText = "纯文本"; // 修改样式 el.style.color = "red"; el.style.fontWeight = "bold"; // 修改属性 el.setAttribute("href", "http://xxx.com"); // 创建/插入节点 var newEl = document.createElement("p"); document.body.appendChild(newEl);
1
静态与动态网页
▶
  • 静态网页：文件扩展名为 .htm / .html
  • 动态网页：内容由服务器动态生成，如 .php / .jsp / .aspx
  • 页面上的滚动字幕（<marquee>）不属于动态网页技术
  • AJAX是动态网页技术，不是静态网页技术（样题第29题易错！）
2
JSP基本概念
▶
  • JSP = HTML + JSP标记 + Java脚本程序
  • Java脚本以 <% 开始，%> 结束
  • JSP标记分为：指令标记和动作标记，两者处理模式不同
  • JSP注释分为：HTML注释（<!-- -->）和隐藏注释（<%-- --%>）
⚠️ 样题第28题：JSP指令标记和动作标记的处理模式是不同的（不是相同的）
3
ASP.NET基本概念
▶
  • ASP.NET 支持多种语言（C#、VB.NET等），不只是C#
  • 页面文件后缀：.aspx
  • 运行在 .NET Framework 上
  • 通常运行在 IIS 服务器上
⚠️ 样题第30题：ASP.NET不是只能用C#，也可用VB.NET等

  简单应用题 · JavaScript 房贷计算器（仿样题34题）

考点：表单取值、Math.pow()、JavaScript计算、DOM操作显示结果

<!DOCTYPE html>
<html>
<head><meta charset="UTF-8"><title>房贷计算器</title></head>
<body>
  <h2>房贷计算器</h2>
  <label>贷款金额（元）：<input type="number" id="amount"></label><br>
  <label>年利率（%）：<input type="number" id="rate" step="0.01"></label><br>
  <label>贷款年限：<input type="number" id="years"></label><br>
  <button onclick="calculate()">计算</button>
  <p id="result"></p>

  <script>
    function calculate() {
      var P = parseFloat(document.getElementById("amount").value);
      var r = parseFloat(document.getElementById("rate").value) / 100 / 12;
      var n = parseInt(document.getElementById("years").value) * 12;
      var monthly = P * r * Math.pow(1+r, n) / (Math.pow(1+r, n) - 1);
      document.getElementById("result").innerHTML =
        "每月还款金额：<b>" + monthly.toFixed(2) + " 元</b>";
    }
  </script>
</body>
</html>
  综合应用题 · 购物车（仿样题35题）

考点：数组操作、DOM动态更新、alert弹窗、综合JavaScript逻辑

<!DOCTYPE html>
<html>
<head><meta charset="UTF-8"><title>购物车</title>
<style>
  .goods-list, .cart-list { width:45%; display:inline-block; vertical-align:top; }
  table { border-collapse:collapse; width:100%; }
  td,th { border:1px solid #ccc; padding:8px; text-align:center; }
</style></head>
<body>
  <h3>商品列表</h3>
  <div class="goods-list">
    <table><tr><th>商品</th><th>价格</th><th>操作</th></tr>
      <tr><td>苹果</td><td>5</td><td><button onclick="addToCart('苹果',5)">加入购物车</button></td></tr>
      <tr><td>香蕉</td><td>3</td><td><button onclick="addToCart('香蕉',3)">加入购物车</button></td></tr>
    </table>
  </div>
  <div class="cart-list">
    <h3>购物车</h3>
    <table><tr><th>商品</th><th>价格</th><th>数量</th></tr>
      <tbody id="cartBody"></tbody>
    </table>
    <p>总价：<b id="total">0</b> 元</p>
    <button onclick="checkout()">结账</button>
  </div>
  <script>
    var cart = {};
    function addToCart(name, price) {
      if(cart[name]) { cart[name].qty++; }
      else { cart[name] = { price: price, qty: 1 }; }
      renderCart();
    }
    function renderCart() {
      var tbody = document.getElementById("cartBody");
      tbody.innerHTML = "";
      var total = 0;
      for(var name in cart) {
        var item = cart[name];
        tbody.innerHTML += "<tr><td>"+name+"</td><td>"+item.price+"</td><td>"+item.qty+"</td></tr>";
        total += item.price * item.qty;
      }
      document.getElementById("total").innerText = total;
    }
    function checkout() { alert("结账成功"); }
  </script>
</body>
</html>
